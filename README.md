## Django 작동법
$ python3 manage.py runserver

## Django Admin 사용법
$ python manage.py runserver 이후 http://localhost:8000/admin 에서 user 를 관리할 수 있다.  
$ python manage.py createsuperuser 명령어를 통해 임의의 super user 를 만들어 관리자페이지에 로그인할 수 있다.
